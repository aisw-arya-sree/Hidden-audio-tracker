# 🔐 Hidden Audio Tracker

A secure web-based tool to hide audio messages inside images using steganography and Django. This project ensures private, encrypted, and undetectable communication.

## 📌 Features

- Text-to-Audio using gTTS
- Audio Steganography with LSB (Least Significant Bit)
- AES Encryption for message privacy
- User Login and Authentication (Django-based)
- Decode audio from image files securely

## 🛠️ Technologies Used

- Python, Django
- gTTS, Pillow, base64
- HTML, CSS, Bootstrap
- MySQL

## 🚀 How to Run the Project

1. Open terminal in the project folder
2. Install required packages:  
   pip install -r requirements.txt
3. Start the Django server:  
   python manage.py runserver
4. Open browser and go to:  
   http://127.0.0.1:8000/
